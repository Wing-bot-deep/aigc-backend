项目背景

我们在一家头部 4A 营销代理商（如蓝色光标）内部，需要自研一套赋能业务员（优化师/阿康）的“AIGC 营销素材智能生成平台”。

目标客群（最终广告主）：维达、汤臣倍健、利洁时等拥有严苛品牌 VI（视觉识别系统）规范的 FMCG（快消）头部 KA 客户。

业务痛点：

传统依靠外部影棚拍摄和外包设计的单图成本过百，且耗时数天，无法满足社媒时代 A/B Test 的海量素材需求。

直接使用 Midjourney 或通用大模型 API 生成素材时，面临两个致命缺陷：“商品包装及 Logo 严重变形”（客户绝对无法接受）以及“文字排版乱码”。

产品定位：一个极简的业务端工作台。前端必须屏蔽所有复杂的 AI 节点连线和参数调整，让毫无技术背景的业务员通过极其简单的输入，就能产出“100% 不形变、符合品牌调性、直接可商用”的图文素材。

产品核心流程

该系统必须包含以下四个核心串联流程：

1. 意图解析与极简输入层 (Input & Intent Parsing)

用户交互：业务员在前端只需选择“品牌（如：得宝Tempo）”、“核心商品（如：手帕纸）”，并输入极其简短的卖点关键词（如：“轻奢、白领、下午茶场景”）。

系统动作：后端调用 LLM（如 GPT-4 / 混元）基于内置的 Prompt 矩阵，将用户简短的输入自动扩写、翻译为精确的文案和底层生图需要的结构化 Prompt（含正向与负向提示词）。

2. LLM 爆款文案生成流 (Copywriting Pipeline)

系统动作：基于步骤 1 的意图，结合对应媒体平台（如小红书/抖音）的“网感”风格指南，生成多套带 Emoji 的文案标题与正文，供用户一键复制或一键发送至投放系统。

3. AIGC 防形变视觉生成流 (Vision Generation Pipeline - 最核心的硬核技术区)

底座路由：引入 Flux.1 与 SDXL 框架作为底层生图引擎（利用 Flux 优秀的文字排版能力和 SDXL 的生态）。

品牌资产挂载 (LoRA)：系统根据用户选择的品牌，自动挂载预先训练好的该品牌专属 LoRA 模型（例如：提取得宝 Tempo 的“莫兰迪色、白领轻奢”视觉元素），确保全局色调和画风契合品牌调性。

边缘约束控制 (ControlNet)：在生图管线中强制引入 ControlNet（如 Canny 或 Depth 预处理），将业务员提前上传的“商品白底图/标准包装图”作为深度条件输入，强控商品包装边缘，确保生成的商品 Logo 与环境光影严丝合缝，达到“0 形变”。

4. 闭环流转与反馈层 (Feedback Loop)

生成的图片支持在前端进行轻量级 Inpainting（局部重绘）修改。

输出的图文素材自动沉淀至企业内容资产库（DAM），并开放 API 接口供内部的“Auto-Bidding 智能出价投放引擎”直接抓取调用。
